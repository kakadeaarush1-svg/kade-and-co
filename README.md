# 📈 Kade & Co. — Paper Trading Simulator

Welcome to **Kade & Co.**, an elegant and luxurious paper trading simulator designed for the sophisticated investor. This application provides a high-end interface to practice trading strategies without real capital risk.

## ✨ Features
- **Luxury UI/UX**: A deep navy and silk white theme with elegant serif typography.
- **Tiger Branding**: Integrated company emblem representing strength and agility.
- **Real-time Simulation**: Interactive trade execution and price quote updates.
- **Portfolio Management**: Track your positions, market value, and P&L in real-time.
- **Interactive Charts**: Visualize price movements with built-in canvas charting.
- **Order History**: Keep a record of all your simulated transactions.

## 🛠️ Built With
- **HTML5**: Structured with semantic elements for a robust layout.
- **CSS3**: Styled with modern CSS variables, glassmorphism, and responsive design.
- **JavaScript (ES6)**: Dynamic client-side logic for a seamless experience.

## 🚀 Getting Started
1. Open `index.html` in your favorite web browser.
2. Enter a ticker symbol in the "Live Quote" section to see current data.
3. Use the "Place Order" card to execute buy or sell trades.
4. Monitor your "My Portfolio" and "Order History" as you trade.
